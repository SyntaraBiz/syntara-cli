# Changelog

Todos los cambios notables de este proyecto se documentarán en este archivo.

El formato está basado en [Keep a Changelog](https://keepachangelog.com/es-ES/1.0.0/),
y este proyecto se adhiere a [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-05-XX

### Added (Añadido)
- Comando `sitemap` para generación automatizada de sitemap.xml mezclando rutas estáticas y dinámicas.
- Comando `pwa-icons` para generar automáticamente los íconos requeridos para PWA a partir de una imagen fuente.
- Comando `optimize-images` para compresión masiva de imágenes (PNG, JPG) a formato WebP optimizado.
- Estructura base del CLI utilizando `commander` y `sharp`.
- Documentación oficial en README.md orientada a consumo público vía `npx`.
